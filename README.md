# ICCW Tech - Custom Software Solutions

Welcome to the official repository for **ICCW Tech**, a SaaS and Custom Software Development agency.

## Overview
This is the frontend source code for the ICCW Tech marketing and business website, built to showcase our industry solutions, CRM/ERP services, and core values. 

The website is engineered for speed, scalability, and an exceptional user experience using modern web technologies.

## Tech Stack
- **Framework:** React.js (v19)
- **Build Tool:** Vite
- **Styling:** Tailwind CSS (v4)
- **Routing:** React Router DOM (v7)
- **Animations:** Framer Motion
- **Icons:** Lucide React

## Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/iccw-tech-website.git
   ```
2. Navigate into the project directory:
   ```bash
   cd iccw-tech-website
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## Deployment
This application is configured for continuous deployment on **Netlify**. 
Pushing to the `main` branch will automatically trigger a production build.

The build configuration and client-side routing fallback are explicitly defined in the `netlify.toml` file.
